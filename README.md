# chat
聊天应用(毕设:智能老人陪伴软件)

预计功能:视频、音频、新闻检索推送, 聊天, 智能陪聊

现完成:注册登录、在线单聊

现有功能待改进:

1.注册登录的验证

2.将未读的消息存储数据库,用户上线时检索未读消息中发给该用户

3.消息窗口的自动滚动
